Potenzfunktionen sind die Bausteine ganzrationaler Funktionen. Das Verständnis ihres Verlaufs, ihrer Symmetrie und ihres Verhaltens im Unendlichen ist Grundlage für alle weiteren Themen.

Potenzfunktionen Teil A + B
f(x) = xⁿ – Grundformen, Symmetrie, Verhalten
▾
Potenzfunktionen haben die Form f(x) = xⁿ mit n ∈ ℤ. Ihr Verlauf hängt stark davon ab, ob n gerade oder ungerade ist.
📖 Vertiefung & Lernmaterial

Gerade Exponenten (n = 2, 4, 6, …)

  • Graph ist achsensymmetrisch zur y-Achse: f(−x) = f(x)
  • U-förmiger Verlauf, Minimum bei x = 0
  • Verhalten: x → ±∞ ⇒ f(x) → +∞

Ungerade Exponenten (n = 1, 3, 5, …)

  • Graph ist punktsymmetrisch zum Ursprung: f(−x) = −f(x)
  • S-förmiger Verlauf, streng monoton steigend
  • Verhalten: x → +∞ ⇒ f(x) → +∞, x → −∞ ⇒ f(x) → −∞

Negative Exponenten

f(x) = x⁻¹ = 1/x → Hyperbel, nicht definiert bei x = 0
💡 Gerade n: Graph kehrt um (Minimum). Ungerade n: Graph schneidet die x-Achse und läuft durch.
Übungsaufgaben
1 Prüfe, ob f(x) = x⁴ − 2x² achsen- oder punktsymmetrisch ist.
f(−x) = (−x)⁴ − 2(−x)² = x⁴ − 2x² = f(x) → achsensymmetrisch zur y-Achse.
2 Beschreibe den Verlauf von f(x) = x⁵ für x → ±∞.
Ungerade, positiver Leitkoeffizient: x → +∞ ⇒ f(x) → +∞ | x → −∞ ⇒ f(x) → −∞.

Eigenschaften analysieren Teil A + B
Definitionsbereich, Wertebereich, Nullstellen, Symmetrie, Verhalten für x → ±∞
▾
Die vollständige Analyse einer Funktion folgt einem festen Schema: Definitionsbereich – Symmetrie – Nullstellen – Verhalten im Unendlichen – Wertebereich.
📖 Vertiefung & Lernmaterial

Definitionsbereich D

Alle x-Werte, für die f definiert ist. Bei Polynomen immer D = ℝ.

Symmetrie prüfen

Achsensymmetrie: f(−x) = f(x)  |  Punktsymmetrie: f(−x) = −f(x)

Nullstellen bestimmen

f(x) = 0 lösen: Ausklammern, Substitution (z = x²), pq-Formel, Satz von Vieta.

pq-Formel: x = −p/2 ± √((p/2)² − q)   für x² + px + q = 0

Wertebereich W

Ergibt sich aus Extremwerten (Kurvenanalyse) und dem Verhalten im Unendlichen.

⚡ Reihenfolge einhalten: D → Symmetrie → Nullstellen → Verhalten → erst dann Extrema bestimmen.
Übungsaufgaben
1 Bestimme die Nullstellen von f(x) = x³ − 4x.
f(x) = x(x² − 4) = x(x−2)(x+2) → Nullstellen: x₁ = 0, x₂ = 2, x₃ = −2
2 Analysiere g(x) = x⁴ − 3x²: Symmetrie, Nullstellen und Verhalten für x → ±∞.
Symmetrie: g(−x) = g(x) → achsensymmetrisch.
Nullstellen: x²(x²−3) = 0 → x = 0 (doppelt), x = ±√3.
Verhalten: Grad 4, aₙ = 1 > 0 → beide Seiten → +∞.
3 Bestimme die Nullstellen von h(x) = x² − 3x + 2 mit der pq-Formel.
p = −3, q = 2 | x = 3/2 ± √(9/4 − 2) = 3/2 ± √(1/4) = 3/2 ± 1/2 → x₁ = 2, x₂ = 1

Spiegelung, Verschiebung, Streckung Teil A + B
Aus f(x) wird g(x) = a · f(b · (x − d)) + c
▾
Jede Transformation lässt sich als Veränderung der Funktionsvorschrift schreiben. Die allgemeine Form ist g(x) = a · f(b · (x − d)) + c.
📖 Vertiefung & Lernmaterial

Spiegelungen

TransformationSchreibweiseWirkung
An der x-Achseg(x) = −f(x)y-Werte werden negiert
An der y-Achseg(x) = f(−x)x-Werte werden negiert

Verschiebungen

g(x) = f(x − d) + c
  • +c → nach oben, −c → nach unten
  • x−d → nach rechts (Achtung: Vorzeichen!), x+d → nach links

Streckungen

g(x) = a · f(b · x)
  • |a| > 1: Streckung in y-Richtung (Stauchung bei 0 < |a| < 1)
  • |b| > 1: Stauchung in x-Richtung
  • Negatives a oder b bewirkt zusätzlich Spiegelung
⚡ Verschiebung in x-Richtung wirkt entgegen der Intuition: f(x − 2) verschiebt nach rechts!

Beispiel

g(x) = −(x+3)² + 1 aus f(x) = x²:
Spiegelung an x-Achse (−), Verschiebung 3 nach links (+3), Verschiebung 1 nach oben (+1).
Übungsaufgaben
1 Wie entsteht g(x) = 2(x − 1)³ + 3 aus f(x) = x³?
Streckung in y-Richtung um Faktor 2 (a=2), Verschiebung 1 nach rechts (d=1), Verschiebung 3 nach oben (c=3).
2 Gib die Funktionsvorschrift für f(x) = x² an, verschoben um 4 nach rechts, 2 nach unten, gespiegelt an der x-Achse.
g(x) = −(x − 4)² − 2
3 Wo liegt der Scheitelpunkt von h(x) = (x + 2)² − 5?
Verschiebung: x+2 → 2 nach links, −5 → 5 nach unten. Scheitelpunkt: S(−2 | −5)

Änderungsraten, Sekante & Tangente Teil A + B
Mittlere und lokale Änderungsrate, graphisches Ableiten
▾
Die mittlere Änderungsrate beschreibt die durchschnittliche Steigung über ein Intervall (Sekante). Die lokale Änderungsrate (Ableitung) ist die momentane Steigung an einem Punkt (Tangente).
📖 Vertiefung & Lernmaterial

Mittlere Änderungsrate (Sekantensteigung)

mₛ = [f(b) − f(a)] / (b − a) = Δy / Δx

Steigung der Geraden durch P(a | f(a)) und Q(b | f(b)).

Lokale Änderungsrate (Tangentensteigung / Ableitung)

f'(x₀) = lim[h→0] [f(x₀ + h) − f(x₀)] / h

Grenzwert der Sekantensteigung, wenn der zweite Punkt gegen x₀ rückt.

Tangentengleichung

t(x) = f'(x₀) · (x − x₀) + f(x₀)

Graphisches Ableiten

Wenn f …dann f' …
steigtist positiv

hat einen Wendepunkthat ein Extremum
💡 Die Tangente berührt den Graphen in genau einem Punkt und hat dort dieselbe Steigung wie f.
Übungsaufgaben
1 Berechne die mittlere Änderungsrate von f(x) = x² auf [1; 4].
mₛ = [f(4) − f(1)] / (4 − 1) = (16 − 1) / 3 = 15/3 = 5
2 Bestimme die Tangentengleichung an f(x) = x³ − x in x₀ = 1.
f'(x) = 3x² − 1 → f'(1) = 2 | f(1) = 0 | t(x) = 2(x − 1) + 0 = 2x − 2
3 f' hat bei x = 2 eine Nullstelle mit Vorzeichenwechsel von − nach +. Was gilt für f?
f hat bei x = 2 ein lokales Minimum. Davor fällt f (f' < 0), danach steigt f (f' > 0).

Ableitungsregeln Teil A + B
Potenz-, Faktor- und Summenregel – alle ganzrationalen Funktionen ableiten
▾
Mit drei Regeln lässt sich jede ganzrationale Funktion ableiten: Potenzregel, Faktorregel, Summenregel. Sie werden in der Klausur immer gebraucht.
📖 Vertiefung & Lernmaterial

Potenzregel

f(x) = xⁿ → f'(x) = n · xⁿ⁻¹

Faktorregel

f(x) = c · g(x) → f'(x) = c · g'(x)

Summenregel

f(x) = g(x) + h(x) → f'(x) = g'(x) + h'(x)

Sonderfälle

  • Konstante: f(x) = c → f'(x) = 0
  • Lineare Funktion: f(x) = mx + b → f'(x) = m

Vollständiges Beispiel

f(x) = 3x⁴ − 2x² + 5x − 1
f'(x) = 12x³ − 4x + 5
f''(x) = 36x² − 4
⚡ Bei Teil A (hilfsmittelfrei) musst du diese Regeln sicher auswendig können und schnell anwenden!
Übungsaufgaben
1 Leite ab: f(x) = 4x³ − 3x² + 7x − 2
f'(x) = 12x² − 6x + 7
2 Leite zweimal ab: g(x) = −x⁴ + 6x²
g'(x) = −4x³ + 12x | g''(x) = −12x² + 12
3 Für welchen Punkt auf f(x) = x² hat die Tangente die Steigung 6?
f'(x) = 2x = 6 → x = 3 | f(3) = 9 → Punkt: (3 | 9)
Mit der Ableitung lässt sich das Verhalten einer Funktion vollständig beschreiben: Wo steigt sie, wo fällt sie, wo liegen Hoch- und Tiefpunkte, wo ändert sich die Krümmung?

Monotonie & Extrempunkte Teil A + B
Wann steigt/fällt f? Lokale und globale Hoch-/Tiefpunkte
▾
Das Vorzeichen von f' entscheidet über Monotonie. Nullstellen von f' sind Kandidaten für Extremstellen – ob es wirklich eine ist, muss mit der 2. Ableitung oder dem Vorzeichenwechsel von f' geprüft werden.
📖 Vertiefung & Lernmaterial

Monotonie

f'(x) > 0 auf (a,b) → f streng monoton steigend f'(x) < 0 auf (a,b) → f streng monoton fallend

Notwendige Bedingung für Extremstellen

f'(x₀) = 0

Achtung: Nicht jede Nullstelle von f' ist eine Extremstelle (z.B. Sattelpunkt)!

Hinreichende Bedingung

BedingungTyp
f''(x₀) > 0lokales Minimum
f''(x₀) < 0lokales Maximum
f''(x₀) = 0keine Aussage → VZW von f' prüfen!

Vorzeichenwechsel von f' (VZW)

  • f' wechselt von + nach −: lokales Maximum
  • f' wechselt von − nach +: lokales Minimum
  • Kein VZW: Sattelpunkt (kein Extremum)

Alle lokalen Extremwerte und den Wertebereich vergleichen. Bei unbeschränktem D = ℝ haben Polynome mit ungeradem Grad keine globalen Extrema.

Punkte, Ortsvektoren & Verbindungsvektoren Nur Teil A
Koordinatisierung des Raumes im ℝ³
▾
Punkte im Raum haben drei Koordinaten. Der Ortsvektor zeigt vom Ursprung zum Punkt. Der Verbindungsvektor AB⃗ zeigt von A nach B und ist die Differenz der Ortsvektoren.
📖 Vertiefung & Lernmaterial

Punkt und Ortsvektor

Punkt P(x|y|z)  →  Ortsvektor OP⃗ = (x, y, z)ᵀ

Verbindungsvektor

AB⃗ = OB⃗ − OA⃗ = (b₁−a₁ | b₂−a₂ | b₃−a₃)ᵀ
Beispiel: A(1|2|3), B(4|0|1) → AB⃗ = (3|−2|−2)ᵀ

Mittelpunkt einer Strecke

M = ½ · (OA⃗ + OB⃗)
Übungsaufgaben
1 Berechne AB⃗ für A(2|0|−1) und B(5|3|2).
AB⃗ = (5−2 | 3−0 | 2−(−1))ᵀ = (3|3|3)ᵀ
2 Bestimme den Mittelpunkt der Strecke AB mit A(0|4|2) und B(6|0|−2).
M = ½ · (0+6 | 4+0 | 2+(−2))ᵀ = (3|2|0)

Komponentenweise rechnen mit Vektoren
▾
Vektoren werden komponentenweise addiert und subtrahiert. Bei der Skalarmultiplikation wird jede Komponente mit der Zahl multipliziert.
📖 Vertiefung & Lernmaterial

Addition und Subtraktion

a⃗ ± b⃗ = (a₁±b₁ | a₂±b₂ | a₃±b₃)ᵀ

Skalarmultiplikation

λ · a⃗ = (λ·a₁ | λ·a₂ | λ·a₃)ᵀ
  • |λ| > 1: Vektor wird länger
  • 0 < |λ| < 1: Vektor wird kürzer

Länge & Kollinearität Nur Teil A
Betrag eines Vektors und Parallelität prüfen
▾
Der Betrag eines Vektors ist seine Länge (Pythagoras im Raum). Zwei Vektoren sind kollinear (parallel), wenn einer ein skalares Vielfaches des anderen ist.
📖 Vertiefung & Lernmaterial

Betrag (Länge)

|v⃗| = √(v₁² + v₂² + v₃²)
Beispiel: v⃗ = (3|4|0)ᵀ → |v⃗| = √(9+16+0) = 5

Einheitsvektor (Länge = 1)

ê = v⃗ / |v⃗|

Kollinearität prüfen

a⃗ und b⃗ sind kollinear, wenn gilt: a⃗ = λ · b⃗ für ein λ ∈ ℝ.

Vorgehen: λ aus erster Komponente berechnen, dann bei allen anderen Komponenten prüfen.

⚡ Wenn auch nur eine Komponente nicht passt: die Vektoren sind nicht kollinear!
Übungsaufgaben
1 Berechne die Länge von v⃗ = (1|2|2)ᵀ.
|v⃗| = √(1+4+4) = √9 = 3
2 Sind a⃗ = (2|−4|6)ᵀ und b⃗ = (1|−2|3)ᵀ kollinear?
λ = 2 aus 1. Komponente. 2·(−2)=−4 ✓ | 2·3=6 ✓ → Ja, kollinear (λ = 2).
3 Sind c⃗ = (1|2|3)ᵀ und d⃗ = (2|4|7)ᵀ kollinear?
λ = 2 (1. Komponente). 2·2=4 ✓ | 2·3=6 ≠ 7 ✗ → Nicht kollinear.
