Definition

∂ Grundverständnis des Ableitungsbegriffs

Mittlere und lokale Änderungsrate, graphisches Ableiten, Sekante und Tangente

📊
Mittlere Änderungsrate
Differenzenquotient – Sekante
▾
Definition
Die mittlere Änderungsrate von f auf [x₀; x₁] entspricht der Steigung der Sekante:
mittlere Änderungsrate = Δy/Δx = (f(x₁) − f(x₀)) / (x₁ − x₀)

Einheit: [y-Einheit] pro [x-Einheit]

Beispiel

f(x) = x² auf [1; 3]:

(f(3) − f(1)) / (3 − 1) = (9 − 1) / 2 = 8/2 = 4

Mittlere Änderungsrate = 4

Übungsaufgaben
Berechne die mittlere Änderungsrate von f(x) = x² auf dem Intervall [2; 4].

Lokale Änderungsrate & Tangente
Ableitung an einer Stelle – f'(x₀)
▾
Kernidee
Wenn man Δx → 0 gehen lässt, wird die Sekante zur Tangente. Die Steigung der Tangente im Punkt x₀ heißt lokale Änderungsrate oder Ableitung:
f'(x₀) = lim[Δx→0] (f(x₀+Δx) − f(x₀)) / Δx
Tangentengleichung
t(x) = f'(x₀) · (x − x₀) + f(x₀)

Steigung m = f'(x₀), Punkt (x₀ | f(x₀)) einsetzen

Graphisches Ableiten – Regeln
  • Steigt f, so ist f' > 0 (liegt über der x-Achse)
  • Fällt f, so ist f' < 0 (liegt unter der x-Achse)
  • Hat f ein Extremum, so hat f' eine Nullstelle
  • Hat f einen Wendepunkt, so hat f' ein Extremum

Monotonie & Extrempunkte
Notwendiges & hinreichendes Kriterium
▾
Monotonie über f'
BedingungBedeutung
f'(x) > 0f ist streng monoton wachsend
f'(x) < 0f ist streng monoton fallend
f'(x) = 0mögliche Extremstelle (prüfen!)
Vorgehen: Extremstellen bestimmen
  1. 1f'(x) bilden
  2. 2f'(x) = 0 lösen → mögliche Extremstellen x₀ (notwendige Bedingung)
  3. 3Vorzeichen von f' links und rechts prüfen oder f''(x₀) berechnen (hinreichende Bedingung)
  4. 4f(x₀) berechnen → Extrempunkt (x₀ | f(x₀)) angeben
Hinreichende Kriterien
  • VZW-Kriterium: f'(x) wechselt von + nach − → Maximum; von − nach + → Minimum
  • 2. Ableitung: f''(x₀) < 0 → Maximum; f''(x₀) > 0 → Minimum; f''(x₀) = 0 → keine Aussage

Ein Sattelpunkt ist ein Wendepunkt mit waagerechter Tangente: f'(x₀) = 0 und f''(x₀) = 0. Es gibt kein Extremum!

Lokale vs. globale Extrema
Unterschied und Ermittlung
▾
Unterschied
  • Lokales Extremum: Höchster/niedrigster Punkt in der Umgebung – gefunden über f'(x₀)=0
  • Globales Extremum: Höchster/niedrigster Punkt auf dem gesamten Definitionsbereich
Globale Extrema auf Intervall [a;b] finden
  1. 1Alle lokalen Extremstellen im Intervall bestimmen
  2. 2Randwerte f(a) und f(b) berechnen
  3. 3Alle Werte vergleichen → größter = globales Maximum, kleinster = globales Minimum
Übungsaufgabe

Koordinatisierungen, Ortsvektoren, Vektoroperationen, Länge, Kollinearität

📍
Punkte & Ortsvektoren
Koordinatensystem im Raum
▾
Definitionen
  • Punkt P(x|y|z): Position im 3D-Raum
  • Ortsvektor ⃗p: Pfeil vom Ursprung O zu P
  • Verbindungsvektor: ⃗AB = ⃗b − ⃗a (Ziel minus Start)
Schreibweise
P = (2 | −1 | 3) → Ortsvektor: p = (2 / −1 / 3) (Spalte)

Im Heft werden Vektoren als Spalte mit drei Einträgen geschrieben.

Übungsaufgaben
A = (1|2|3) und B = (4|0|1). Berechne den Verbindungsvektor ⃗AB.
➕
Vektoroperationen
Addition, Subtraktion, Skalarmultiplikation
▾
Rechenregeln (komponentenweise)
a = (a₁/a₂/a₃), b = (b₁/b₂/b₃) a + b = (a₁+b₁ / a₂+b₂ / a₃+b₃) a − b = (a₁−b₁ / a₂−b₂ / a₃−b₃) λ · a = (λ·a₁ / λ·a₂ / λ·a₃)

Länge & Kollinearität
Betrag eines Vektors – parallele Vektoren
▾
Betrag (Länge) eines Vektors
|a⃗| = √(a₁² + a₂² + a₃²) Beispiel: a⃗ = (1/2/2) |a⃗| = √(1 + 4 + 4) = √9 = 3
Kollinearität
Zwei Vektoren a⃗ und b⃗ heißen kollinear, wenn es ein λ ∈ ℝ gibt mit:
b⃗ = λ · a⃗

Das bedeutet: b ist ein Vielfaches von a. Alle Komponenten haben dasselbe Verhältnis.

Kollinearität prüfen
