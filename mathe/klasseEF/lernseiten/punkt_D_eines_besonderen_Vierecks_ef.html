Den Punkt D bestimmen

Einführungsphase · Analytische Geometrie

Lehrervorführung – schau zu
Rechteck: Punkt D bestimmen
Gegeben sind drei Punkte. Schau zu – klappe die Schritte auf.
\(A(0|0|0)\) \(B(4|2|0)\) \(C(5|0|3)\)
1
Was muss erfüllt sein?
▾

Ein Rechteck \(ABCD\) erfüllt:

  • ∥Zwei Paar parallele Seiten: \(\overrightarrow{AB} = \overrightarrow{DC}\) und \(\overrightarrow{AD} = \overrightarrow{BC}\)
  • ⚖️Gegenüber­liegende Seiten gleich lang: \(|AB| = |DC|\) und \(|AD| = |BC|\)
  • 📐Rechte Winkel: \(\overrightarrow{AB} \cdot \overrightarrow{AD} = 0\) – geprüft mit Pythagoras an der Diagonale

Aus \(\overrightarrow{AB} = \overrightarrow{DC}\) folgt: \(D = C - \overrightarrow{AB}\) oder gleichwertig \(D = A + \overrightarrow{BC}\).

2
Prüfen: Ist ein rechter Winkel bei B möglich?
▾

Dafür müssten \(\overrightarrow{BA}\) und \(\overrightarrow{BC}\) senkrecht aufeinander stehen.

Der elegante Weg wäre das Skalarprodukt – das kommt aber erst in der Q1. 😄

Wir nutzen stattdessen den Satz des Pythagoras als Probe:

▶
Pythagoras als Rechtwinkligkeitsprobe aufklappen
▾

Wenn bei \(B\) ein rechter Winkel liegt, muss die Diagonale \(AC\) die Hypotenuse sein. Dann gilt:

\(|AC|^2 = |AB|^2 + |BC|^2\)

Wir berechnen die drei Längen:

\(\overrightarrow{AB} = \begin{pmatrix}4\\2\\0\end{pmatrix} \Rightarrow |AB|^2 = 16+4+0 = 20\)
\(\overrightarrow{BC} = C - B = \begin{pmatrix}1\\-2\\3\end{pmatrix} \Rightarrow |BC|^2 = 1+4+9 = 14\)
\(\overrightarrow{AC} = C - A = \begin{pmatrix}5\\0\\3\end{pmatrix} \Rightarrow |AC|^2 = 25+0+9 = 34\)

Probe:

\(|AB|^2 + |BC|^2 = 20 + 14 = 34 = |AC|^2 \checkmark\)

Der Pythagoras ist erfüllt – bei \(B\) liegt ein rechter Winkel. Ein Rechteck ist möglich. ✓

3
Punkt D aus den Vektoren herleiten
▾

Im Parallelogramm (und damit auch im Rechteck) gilt:

\(\overrightarrow{OD} = \overrightarrow{OA} + \overrightarrow{BC}\)

Das bedeutet: \(D\) entsteht, indem man von \(A\) aus denselben Schritt macht wie von \(B\) nach \(C\).

\(\overrightarrow{BC} = \begin{pmatrix}1\\-2\\3\end{pmatrix}\)
\(\overrightarrow{OD} = \begin{pmatrix}0\\0\\0\end{pmatrix} + \begin{pmatrix}1\\-2\\3\end{pmatrix} = \begin{pmatrix}1\\-2\\3\end{pmatrix}\)

Also: \(D(1|-2|3)\)

Gleichwertig wäre auch: \(\overrightarrow{OD} = \overrightarrow{OC} + \overrightarrow{BA} = \begin{pmatrix}5\\0\\3\end{pmatrix} + \begin{pmatrix}-4\\-2\\0\end{pmatrix} = \begin{pmatrix}1\\-2\\3\end{pmatrix}\) ✓

4
Vektoren bestätigen & Längen berechnen
▾

Wir prüfen, ob \(\overrightarrow{AB} = \overrightarrow{DC}\) gilt:

\(\overrightarrow{AB} = \begin{pmatrix}4\\2\\0\end{pmatrix}, \quad \overrightarrow{DC} = C - D = \begin{pmatrix}5\\0\\3\end{pmatrix} - \begin{pmatrix}1\\-2\\3\end{pmatrix} = \begin{pmatrix}4\\2\\0\end{pmatrix} \checkmark\)

Seitenlängen:

\(|AB| = \sqrt{4^2+2^2+0^2} = \sqrt{20}\)
\(|BC| = \sqrt{1^2+(-2)^2+3^2} = \sqrt{1+4+9} = \sqrt{14}\)

Pythagoras an der Diagonale \(\overrightarrow{AC}\):

\(|\overrightarrow{AC}|^2 = 5^2+0^2+3^2 = 34 = 20 + 14 = |AB|^2 + |BC|^2 \checkmark\)

✅ \(ABCD\) mit \(D(1|-2|3)\) ist ein Rechteck.

Deine Aufgabe
Trapez: Wähle D selbst
Gegeben sind drei Punkte. Bestimme D so, dass ein Trapez entsteht.
\(A(0|0|0)\) \(B(6|0|0)\) \(C(7|3|2)\)
1
Was muss erfüllt sein?
▾

Ein Trapez \(ABCD\) mit \(AB \parallel DC\) erfüllt:

  • ∥\(\overrightarrow{DC} = k \cdot \overrightarrow{AB}\) für ein \(k > 0\), \(k \neq 1\)
  • 📍\(D\) ergibt sich dann aus: \(\overrightarrow{OD} = \overrightarrow{OC} - \overrightarrow{DC} = \overrightarrow{OC} - k \cdot \overrightarrow{AB}\)

Du kannst \(k\) frei wählen – jedes \(k > 0\) mit \(k \neq 1\) ergibt ein Trapez.

2
\(\overrightarrow{AB}\) berechnen und \(k\) wählen
▾

Berechne zuerst \(\overrightarrow{AB}\):

\(\overrightarrow{AB} = B - A = \begin{pmatrix}6\\0\\0\end{pmatrix}\)

Wähle deinen Faktor \(k\):

\(\frac{1}{2}\)
DC ist halb so lang wie AB

Damit ergibt sich \(\overrightarrow{DC}\):

\(\overrightarrow{DC} = \frac{1}{2} \cdot \begin{pmatrix}6\\0\\0\end{pmatrix} = \begin{pmatrix}3\\0\\0\end{pmatrix}\)
3
Punkt D berechnen
▾

Es gilt: \(\overrightarrow{OD} = \overrightarrow{OC} - k \cdot \overrightarrow{AB}\)

Trage dein D ein – für dein gewähltes \(k\).

\(D\;\big(\) | | \(\big)\)
\(\overrightarrow{OD} = \overrightarrow{OC} - k \cdot \overrightarrow{AB}\)
Für \(k = \frac{1}{2}\): \(\overrightarrow{OD} = \begin{pmatrix}7\\3\\2\end{pmatrix} - \begin{pmatrix}3\\0\\0\end{pmatrix} = ?\)
4
Kollinearität und Längen bestätigen
▾

Berechne \(|\overrightarrow{AB}|\) und \(|\overrightarrow{DC}|\) und zeige, dass \(\overrightarrow{DC} = k \cdot \overrightarrow{AB}\).

Trage die Zahlen unter den Wurzeln ein.

\(|\overrightarrow{AB}| = \sqrt{\vphantom{0}}\)
\(|\overrightarrow{DC}| = \sqrt{\vphantom{0}}\)
\(|\overrightarrow{AB}| = \sqrt{6^2+0^2+0^2} = \sqrt{36}\)
Für \(k=\frac{1}{2}\): \(|\overrightarrow{DC}| = \sqrt{3^2+0^2+0^2} = \sqrt{9}\)
✅ Nachweis vollständig! \(ABCD\) ist ein Trapez mit \(AB \parallel DC\).
Deine Aufgabe
Parallelogramm: Punkt D bestimmen
Gegeben sind drei Punkte. Bestimme D so, dass ein Parallelogramm mit \(AB \parallel DC\) entsteht.
\(A(0|0|0)\) \(B(6|0|0)\) \(C(7|3|2)\)
1
Was muss erfüllt sein?
▾

Ein Parallelogramm \(ABCD\) mit \(AB \parallel DC\) erfüllt:

  • ∥\(\overrightarrow{DC} = \overrightarrow{AB}\) – gleiche Richtung, gleiche Länge (\(k = 1\))
  • ∥Damit auch: \(\overrightarrow{AD} = \overrightarrow{BC}\)
  • 📍\(D\) ergibt sich aus: \(\overrightarrow{OD} = \overrightarrow{OA} + \overrightarrow{BC}\) oder gleichwertig \(\overrightarrow{OD} = \overrightarrow{OC} - \overrightarrow{AB}\)
2
Alle Vektoren berechnen
▾

Berechne \(\overrightarrow{AB}\) und \(\overrightarrow{BC}\).

\(\overrightarrow{AB} = B - A\) und \(\overrightarrow{BC} = C - B\) – komponentenweise subtrahieren.
3
Punkt D berechnen
▾

Nutze: \(\overrightarrow{OD} = \overrightarrow{OA} + \overrightarrow{BC}\)

\(D\;\big(\) | | \(\big)\)
\(\overrightarrow{OA} = \begin{pmatrix}0\\0\\0\end{pmatrix}\), \(\overrightarrow{BC} = \begin{pmatrix}1\\3\\2\end{pmatrix}\)
\(\overrightarrow{OD} = \overrightarrow{OA} + \overrightarrow{BC} = ?\)
4
Nachweis: \(\overrightarrow{AB} = \overrightarrow{DC}\)
▾

Berechne \(\overrightarrow{DC}\) mit deinem \(D\) und prüfe, ob \(\overrightarrow{DC} = \overrightarrow{AB}\).

\(\overrightarrow{DC} = C - D\). Mit \(D(1|3|2)\) gilt: \(\overrightarrow{DC} = \begin{pmatrix}7\\3\\2\end{pmatrix} - \begin{pmatrix}1\\3\\2\end{pmatrix} = ?\)
✅ Nachweis vollständig! \(ABCD\) ist ein Parallelogramm.

Alternative: \(AD \parallel BC\) statt \(AB \parallel DC\)

Wenn stattdessen \(AD \parallel BC\) gefordert wird, gilt \(\overrightarrow{AD} = \overrightarrow{BC}\) – also dieselbe Formel, aber mit vertauschter Rolle. Man erhält:

\(\overrightarrow{OD} = \overrightarrow{OC} + \overrightarrow{AB} = \begin{pmatrix}7\\3\\2\end{pmatrix} + \begin{pmatrix}6\\0\\0\end{pmatrix} = \begin{pmatrix}13\\3\\2\end{pmatrix}\)

Also \(D(13|3|2)\) als zweite Lösung.

Probe: \(\overrightarrow{DC} = C - D = \begin{pmatrix}7\\3\\2\end{pmatrix} - \begin{pmatrix}13\\3\\2\end{pmatrix} = \begin{pmatrix}-6\\0\\0\end{pmatrix} = -1 \cdot \overrightarrow{AB}\) – also antiparallel, auch ein Parallelogramm.

F2) Sind \(\vec{u} = \begin{pmatrix}4\\-2\\6\end{pmatrix}\) und \(\vec{v} = \begin{pmatrix}2\\-1\\3\end{pmatrix}\) kollinear? Welcher Faktor \(k\)?

P1) Gegeben: \(A(1|0|2)\), \(B(4|1|2)\), \(C(5|3|4)\).

Bestimme alle drei möglichen Punkte \(D_1\), \(D_2\), \(D_3\), die mit \(A\), \(B\), \(C\) ein Parallelogramm bilden.

Es gibt genau drei verschiedene Möglichkeiten je nachdem, welches Paar von Seiten parallel ist.

\(D_1\big(\) | | \(\big)\)
\(D_2\big(\) | | \(\big)\)
\(D_3\big(\) | | \(\big)\)
Option 1: \(AB \parallel DC\) → \(D = A + \overrightarrow{BC}\)
Option 2: \(AC \parallel DB\) → \(D = A + \overrightarrow{CB}\) … oder denke an die Mittelformel:
Die drei Möglichkeiten sind \(A+C-B\), \(A+B-C\) und \(B+C-A\).
