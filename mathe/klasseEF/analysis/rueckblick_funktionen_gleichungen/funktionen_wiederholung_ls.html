1 · Was ist eine Funktion?

x → Funktion f → f(x)
Funktion: Eine Funktion \(f\) ist eine Zuordnung, die jedem Element \(x\) des Definitionsbereichs \(D_f\) genau ein Element \(f(x)\) des Wertebereichs \(W_f\) zuordnet. Die Punkte \(P(x|f(x))\) bilden den Graphen von \(f\).

Verschiedene Schreibweisen

\(f(x) = 4\cdot x + 7\)  (Funktionsgleichung) – „f von x ist gleich \(4\cdot x+7\)“
\(f : x \rightarrow 4\cdot x + 7\)  (Funktionsvorschrift) – „x wird zu \(4\cdot x+7\)“
In allen Fällen ist \(x\) der Name der Variablen – der Funktionsname kann auch \(g, h, d, \dots\) sein.

Wichtig beim Einsetzen: \(x = 3\) einsetzen bei \(f(x) = 4\cdot x + 7\): \(f(3) = 4\cdot 3 + 7 = 19\). Der eingesetzte Wert ersetzt jedes \(x\) im Term.

4 · Quadratische Funktionen

Normalform: \(f(x) = a\cdot x^2 + b\cdot x + c\)
Scheitelpunktform: \(f(x) = a\cdot(x-d)^2 + e\), Scheitelpunkt \(S(d|e)\)
Faktorisierte Form: \(f(x) = a\cdot(x-r)\cdot(x-s)\), mit Nullstellen \(r\) und \(s\)
Der Streckfaktor \(a\) bestimmt die Öffnung: für \(a>0\) öffnet die Parabel nach oben, für \(a<0\) nach unten. Je größer \(|a|\), desto schmaler die Parabel.

5 · Scheitelpunktform ⇄ Normalform

Beide Formen beschreiben dieselbe Parabel – man kann jederzeit zwischen ihnen wechseln:
Normalform → Scheitelpunktform: quadratische Ergänzung.
Scheitelpunktform → Normalform: ausmultiplizieren (binomische Formel).

Beispiel 1: Normalform → Scheitelpunktform

Gegeben: \(f(x) = x^2 + 6\cdot x + 5\). Tippe dich Schritt für Schritt durch die quadratische Ergänzung.

Schritt 1: Nur die ersten beiden Summanden betrachten: \(x^2+6\cdot x\).
Schritt 2: Ergänze zum vollständigen Quadrat. Die Hälfte von 6 ist 3, also brauchst du \(+3^2\). Damit sich der Wert nicht ändert, ziehst du \(3^2\) gleich wieder ab: \(x^2+6\cdot x+3^2-3^2+5\).
Schritt 3: Die ersten drei Summanden bilden ein Binom: \((x+3)^2-3^2+5\).
Schritt 4: Zusammenfassen: \((x+3)^2-9+5 = (x+3)^2-4\). Ergebnis: \(f(x)=(x+3)^2-4\), Scheitelpunkt \(S(-3|-4)\).

Beispiel 2: Scheitelpunktform → Normalform

Gegeben: \(f(x) = 2\cdot(x-1)^2 + 3\). Tippe dich Schritt für Schritt durchs Ausmultiplizieren.

Schritt 1: Binom ausmultiplizieren: \((x-1)^2 = x^2-2\cdot x+1\).
Schritt 2: Mit dem Streckfaktor 2 multiplizieren: \(2\cdot(x^2-2\cdot x+1) = 2\cdot x^2-4\cdot x+2\).
Schritt 3: Die 3 addieren: \(2\cdot x^2-4\cdot x+2+3\). Ergebnis: \(f(x)=2\cdot x^2-4\cdot x+5\).

Übertrage \(f(x) = x^2 - 8\cdot x + 10\) in die Scheitelpunktform \(f(x) = (x-d)^2 + e\) (der Streckfaktor ist bereits 1, du musst nur \(d\) und \(e\) bestimmen).

\(f(x) = x^2-8\cdot x+16-16+10 = (x-4)^2-6\).
Also \(d=4\) und \(e=-6\).

Tipp 1: Scheitelpunkt \(S(3|5)\), also \(f(x)=a\cdot(x-3)^2+5\).
